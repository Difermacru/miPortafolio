AppSalon

Esta aplicación de citas para cortes de cabello fue desarrollada utilizando tecnologías modernas como Vue.js, TailwindCSS, Pinia y Express. Su frontend está conectado a MongoDB, permitiendo una gestión eficiente de los datos. Además, se diseñaron y desarrollaron REST APIs para manejar las interacciones entre el cliente y el servidor, asegurando una comunicación fluida. Todas las API fueron rigurosamente probadas con Postman para garantizar su correcto funcionamiento. Para la creación y validación de formularios, se implementó FormKit, optimizando la experiencia del usuario y la gestión de datos en la aplicación.
